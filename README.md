# Wheat-Weed-Dataset
The dataset contains 4664 RGB annotated images of weeds in wheat, distributed as follows:  Thornapple (Datura stramonium),  Green foxtail (Setaria viridis),  Perennial ryegrass (Lolium perenne),  Bindweed (Convolvulus),  Wild oats (Avena fatua),  Opium poppy (Papaver somniferum),  Common wheat (Triticum aestivum) - varieties Geya1 and Sashez.
